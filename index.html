Galacticolour

A two dimensional level based game hosted entirely on the web. Developed completely from scratch using vanilla JavaScript with custom animations, movement, controls, enemy AI, menu screens, and game screens. Developed in a 3 person team.

Big Brain Time

A task manager that allows you to easily keep track of tasks and assignments, with implemented study sessions to use when working on said tasks. Also implements a calendar to keep track of due dates and upcoming tasks/events. Built with a custom module that allows for easy placement of elements such as buttons and content sections with full responsiveness to screen size changes.

Battleship

Implementation of the classic game Battleship using an Arduino. The game is displayed on a 32x16 LED display and includes PvP multiplayer and PvE with multiple AI difficulties. Developed in a 5 person team.
